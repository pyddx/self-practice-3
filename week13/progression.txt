เพื่อจัดการเรื่องเวลา–วันที่ เช่น การอ่านเวลาปัจจุบัน, ห่างกันกี่วัน, หาเลขวันในสัปดาห์, บวกวันเพิ่ม, แสดงผลแบบไทย และทดลองทำ logic

1. showCurrentDateTime() — ดูวันเวลาปัจจุบัน
Core Concept: เริ่มต้นด้วย new Date() เพื่อ Snapshot เวลา ณ วินาทีนั้น
Deconstruct: ข้อมูลที่ได้มาเป็นก้อน Object ต้องใช้คำสั่งแยกชิ้นส่วน (Getters) ออกมา
.getFullYear(): ได้ปี ค.ศ. เต็มๆ (เช่น 2025)
.getMonth(): ระวัง! มันเริ่มนับที่ 0 (ม.ค. = 0, ธ.ค. = 11) ต้องบวก 1 เสมอถ้าจะโชว์
.getDate(): วันที่ (1-31)
.getHours(), .getMinutes(), .getSeconds(): ส่วนของเวลา

2. หาความห่างของวัน (Date Difference)
Input: รับค่าวันที่ 2 ตัว (มักจะเป็น String เช่น "2025-01-01") ต้องแปลงเป็น new Date() ก่อน
The Magic: เมื่อเอา Date Object มาลบกัน (date2 - date1) ผลลัพธ์จะเป็น Milliseconds (ms)
Calculation: ต้องรู้ค่าคงที่เพื่อแปลง ms กลับเป็นวัน
สูตร: ผลต่าง ms / (1000 ms * 60 วินาที * 60 นาที * 24 ชั่วโมง) = จำนวนวัน

3. วันนี้วันอะไร? (Day of Week)
Logic: คำสั่ง .getDay() คืนค่าเป็นตัวเลข 0-6 ไม่ใช่ชื่อวัน (0 = อาทิตย์, 1 = จันทร์ ... 6 = เสาร์)
Mapping: เราต้องสร้าง Array เก็บชื่อวันไว้เอง เช่น const days = ["อาทิตย์", "จันทร์", ...]
Usage: ใช้เลขที่ได้จาก getDay() เป็น Index เพื่อหยิบชื่อวันจาก Array มาแสดง (days[index])
4. addDays(date, days) — บวกวันเพิ่ม
Logic: ใช้คู่หู getDate() (อ่านค่าเดิม) และ setDate() (ตั้งค่าใหม่)
Auto-Correction: ระบบ Date ของ JS ฉลาดมาก ให้เอา ค่าวันเดิม + จำนวนที่ต้องการ ได้เลย
ตัวอย่าง: ถ้าวันนี้วันที่ 31 มกรา แล้วบวกไป 1 วัน (setDate(32)) JS จะรู้เองว่าต้องทดไปเป็น 1 กุมภาพันธ์ โดยอัตโนมัติ ไม่ต้องเขียน if เช็คเดือนเอง

5. แสดงวันที่แบบไทย (Thai Formatting)
Easy Way: ใช้ฟังก์ชันสำเร็จรูป .toLocaleDateString("th-TH", options) จะได้ชื่อเดือนภาษาไทยและปี พ.ศ. ทันที
Manual Way: ถ้าต้องคำนวณเอง ให้จำสูตร ปี พ.ศ. = ปี ค.ศ. (.getFullYear()) + 543

6. countdown() — ระบบนับถอยหลัง
Process: คำนวณหา gap = วันเป้าหมาย - เวลาปัจจุบัน
Conversion: เปลี่ยนเศษ gap (ms) ให้เป็นหน่วยที่มนุษย์อ่านรู้เรื่อง
ใช้ Math.floor() ปัดเศษทิ้งเพื่อให้ได้เลขจำนวนเต็ม
ใช้ การหาร (/) เพื่อหาหน่วยใหญ่ (วัน/ชั่วโมง)
ใช้ Modulo (%) เพื่อหาเศษเหลือ ส่งต่อไปคิดหน่วยที่เล็กลง (นาที/วินาที)
